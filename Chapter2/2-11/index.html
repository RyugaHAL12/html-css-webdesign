Formの作り方

ここにフォームの部品が入る


1行テキスト入力欄の作り方

名前：

inputタグ:検索ボックスやテキスト入力エリアを作れる

type属性:textやemailを入れることで入力フォーマットを指定できる

placeholder属性:テキストボックスにあらかじめ文字を入力しておける

入力欄に最初からテキストを表示するには？

→placeholder属性に最初に表示したいテキストを入れる

いろんな種類の１行テキスト入力欄

text：１行のテキスト(初期値)

search：検索する際のテキスト

email：メールアドレスを入力する際のテキスト

tel：電話番号を入力する際のテキスト

url：URLを入力する際のテキスト


ラジオボタンの作り方

性別:男 女 その他

type="radio":radio属性を指定すると、ラジオボタンを作成できる

name:属性に同じ名前を指定することで、そのラジオボタンでグループを作成できる

同じグループのラジオボタンは１つしか選択できなくなる

チェックボックスを作る

赤 青 黄 緑

input type=checkbox → チェックボックスを指定

name属性でグループ化

value属性で値を設定する

送信ボタンを作る

input type=submit → 送信ボタンを作成

value : ボタンの中にいれる文字列

type=image →画像が選択できるようになる。

画像はsrc属性で指定する(ほかと一緒)

セレクトボックスを作る

selectタグ+optionタグで作る

選択肢全体をselectタグで囲み、選択項目はoptionタグで囲む

selectタグの主な属性

name：セレクトボックスの名前

multiple：複数選択が可能かどうか

optionタグの主な属性

value：送信される選択肢の値

selected：最初から選択された状態にする時に指定する

複数行テキストの入力欄を作るtextareaタグ

textareaタグで囲んだ部分は入力欄をクリックしても消えない

デフォルトで表示したいテキストはplaceholder属性で指定するといい。

tips:フォームを動作させるにはPHPでのプログラミングが必要(バックエンド構築)
